### Simple Blockchain Web Application
It's a blockchain based simple web application made with Fastapi Python where user can create transactions and mine block.
![image](https://github.com/user-attachments/assets/7a007c9d-8010-443a-b2fa-f667f2ac8855)
Clone the repository
```
git clone https://github.com/Ahnaf-nub/Blockchain.git
cd Blockchain
```
Install the requirements
```
pip install requirements.txt
```
To run the file locally.
```
python blockchain.py
